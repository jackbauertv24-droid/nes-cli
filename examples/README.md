# NES CLI Examples

This folder contains example outputs generated by nes-cli using Super Mario Bros. 3 (USA) ROM.

## Files

| File | Size | Description |
|------|------|-------------|
| `smb3_intro.gif` | ~3.5 MB | 10-second GIF animation (600 frames, title screen) |
| `smb3_audio.wav` | ~958 KB | 10-second audio (boot sound, then silence) |
| `smb3_game.png` | ~12 KB | Screenshot at frame 600 (title screen) |
| `smb3_gameplay.png` | ~11 KB | Screenshot after pressing START |
| `smb3_ascii.txt` | ~15 KB | ASCII art screenshot |

## Sprites

### Title Screen Sprites (`sprites/`)

```
sprites/
├── chr/
│   ├── spritesheet_gray.png     (64×256, 8×32 grid of 256 tiles)
│   ├── spritesheet_palette0.png  (game palette 1)
│   ├── spritesheet_palette1.png  (game palette 2)
│   ├── spritesheet_palette2.png  (game palette 3)
│   ├── spritesheet_palette3.png  (game palette 4)
│   ├── gray/tile_XXX.png         (256 individual 8×8 tiles)
│   ├── palette0/...
│   ├── palette1/...
│   ├── palette2/...
│   └── palette3/...
├── oam.json                      (metadata for 64 sprites)
└── oam/
    └── spritesheet.png           (visible sprites grid)
```

### Gameplay Sprites (`sprites_gameplay/`)

Same structure, captured during gameplay (different palette colors).

## Output Specifications

- **GIF**: 256×240 pixels, 60fps, ~3.5MB for 10 seconds
- **WAV**: 44100Hz, 16-bit PCM, mono, ~958KB for 10 seconds
- **PNG**: 256×240 pixels, RGBA, authentic NES palette colors
- **CHR spritesheets**: 64×256 pixels (8×32 grid of 8×8 tiles)
- **OAM spritesheet**: Grid of currently visible sprites

## Audio Note

The recorded audio captures SMB3's boot sound (~0.5 seconds), then silence. This is expected - the title screen has no continuous music after the boot sound. To capture gameplay music, press START before recording.

## Sprite Extraction

CHR-ROM sprites use the game's current sprite palettes (4 palettes × 256 tiles). OAM sprites are extracted from the PPU's sprite memory with their actual positions, colors, and flip attributes.

## Note

These examples were generated using a legally obtained ROM for demonstration purposes. Always respect copyright laws when using nes-cli.
